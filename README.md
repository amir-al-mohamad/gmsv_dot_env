# gmsv_dot_env

Реализация .env для Garry's Mod.

В отличии от [gmsv_dot_env](https://github.com/owainjones74/gmsv_dot_env) можно получить внешние переменные окружения.

## Пример использования

```lua
  require("dot_env")

  print(env("TEST"))
```
